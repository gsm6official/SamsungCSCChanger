## ✨ Features

*   **🚫 No Root Required:** Modifies system CSC through safe, authorized channels.
*   **🔒 Knox 0x0:** Guarantees your warranty bit remains untripped.
*   **💾 Zero Data Loss:** Performs a seamless "hot-swap" of the CSC. Your apps, settings, and files stay intact.
*   **🤖 Intelligent Feature Detection:** The dropdown menu clearly shows which CSCs enable features like Call Recording (📞) and Samsung Pay (💳).
*   **⚠️ Built-in Safety Warnings:** Operation logs and visual indicators warn you about CSCs that might require a data wipe on certain models.
*   **🔄 In-App Updater:** Keeps the CSC database up-to-date without needing to download a new version.
*   **📡 Dual Connection Modes:** Works flawlessly over both MTP and ADB connections.

## ⚙️ Quick Start

1.  **Prerequisites:** Ensure you have a Windows PC and the [Samsung USB Drivers](https://gsm6.com/samsung-android-usb-drivers-download-link/) installed (installing Smart Switch is the easiest way).
2.  **Download:** Get the latest `Samsung_CSC_Changer_V2.12.112025.zip` from the [Releases](https://github.com/gsm6official/SamsungCSCChanger/releases) page.
3.  **Prepare Phone:**
    *   Enable `Developer Options` (tap `Build Number` in `Settings > About Phone` 7 times).
    *   Enable `USB Debugging` in `Developer Options`.
    *   Set USB mode to `File Transfer (MTP)`.
4.  **Run:** Extract and run the executable (right-click, select "Run as administrator").
5.  **Connect:** Plug in your phone. Allow USB debugging on the phone prompt.
6.  **Read Info:** Click `Read Device Information` to establish the connection.
7.  **Select & Change:** Choose your desired CSC from the dropdown (look for 📞/💳 icons!) and click `Change CSC`. Your phone will reboot.
